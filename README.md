## Installation

1. Clone the repository:

```bash
    git clone [repo name]
```

2. Navigate into the project directory:

```
    cd [repo name]
```

3. Install the dependencies:

```
   npm install
```

4. Start the development server:

```
   npm run dev
```

Now you can open your browser and visit localhost:3000 to view the project.
